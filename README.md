# Worklog Daily 🚀

Projeto pessoal em desenvolvimento para registrar tarefas de desenvolvimento de forma automatizada.

A ideia é simples: enviar tarefas pelo Telegram, salvar no banco de dados e gerar resumos organizados com IA.

---

## ⚙️ Tecnologias

- Java + Spring Boot
- PostgreSQL
- Integração com API do Telegram
- Integração com IA (Gemini)
- GitHub API

---

## 📅 Como funciona

- Registro de tarefas via bot no Telegram
- Armazenamento no banco de dados
- Geração de resumo diário com IA
- Envio automático para o GitHub
